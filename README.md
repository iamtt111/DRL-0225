# Personal Experience | Live Clock

A premium, single-page personal dashboard featuring a real-time digital clock, full date display, and a modern aesthetic.

You can view the live site here: [https://iamtt111.github.io/DRL-0225/](https://iamtt111.github.io/DRL-0225/)

## ✨ Features
- **Live Digital Clock**: Updates every second with a sleek glow effect.
- **Full Date Display**: Shows the weekday, month, and year.
- **Premium Design**: 
    - **Glassmorphism**: Frosted glass UI elements.
    - **Animated Background**: Dynamic radial gradients.
    - **Responsive**: Fully optimized for mobile and desktop.
- **Modern Tech Stack**: Built with pure HTML5, CSS3, and Vanilla JavaScript.

## 🛠️ Components
- `index.html`: Structure of the personal workspace.
- `index.css`: Custom animations and premium styling.
- `script.js`: Real-time clock and date logic.

## 📝 Customization
To personalize the site, open `index.html` and update the `<h1>` tag with your name:
```html
<h1>YOUR NAME</h1>
```
